<h2>🛠️ Installation Steps :</h2>

<p>1. Clone Repository</p>

```
git clone https://github.com/mdzakied/Quotely.git
```

<p>2. NPM Install</p>

```
npm install
```

<p>3. Run Project</p>

```
npm start-dev
```
